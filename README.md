# SPU Finder 🔎 - ระบบแจ้งของหายได้คืน ม.ศรีปทุม

ระบบประกาศของหาย–ได้คืนสำหรับนักศึกษาและบุคลากร มหาวิทยาลัยศรีปทุม

**ใช้งานได้ที่: [finder-spu.vercel.app](https://finder-spu.vercel.app/)**

---

## 🌟 ฟีเจอร์หลัก (Main Features)

* **ประกาศของหาย:** สร้างโพสต์เพื่อแจ้งติดตามของที่ทำหาย
* **แจ้งของที่เก็บได้:** ประกาศของที่เก็บได้เพื่อคืนให้เจ้าของ
* **ระบบค้นหา:** ค้นหาสิ่งของจากชื่อ หรือรายละเอียด
* **ระบบกรองการค้นหา:** กรองการค้นหาได้ตาม:
    * **ประเภท:** (บัตร/เอกสาร, อิเล็กทรอนิกส์, เสื้อผ้า, ของใช้ส่วนตัว, อื่นๆ)
    * **สถานะ:** (ตามหา, พบแล้ว, ส่งคืนแล้ว)
    * **เวลา:** (ใหม่สุด, เก่าสุด)
* **การจัดการโพสต์:** เจ้าของโพสต์สามารถ แก้ไข, ลบ, หรืออัปเดตสถานะโพสต์ได้
